# MagicWords
Ce projet est une implémentation en Prolog d'un résolveur et d'un générateur pour un casse-tête combinatoire basé sur des mots magiques. Le défi consiste à manipuler des suites d'entiers représentant des mots, en appliquant des règles de réécriture pour transformer ces mots ou générer de nouveaux mots magiques.
